Introduction to CSS - Lesson Summary

This course is based on the Kids Learning Code CSS Tutorial.

  • Day One - July 17, 2020 (Slides 1 to 9)

    We learned about what CSS means and why it is more widely used for styling web pages and elements. We were able to add style tags and set colors for different sections. We will be working on playing around with font styles next.

  • Day Two - July 31, 2020 (Slides 10 to 20)

    We explored a variety of style properties and tried formatting our images using CSS. We also learned about using classes to assign styles to even more specific sections of the HTML body.

  • Day Three - August 7, 2020 (Slides 21 to 50)

    Today we dove into the "box model" and learned about margins and padding. We also started using section tags to divide our page and apply different styles to each area, and also learned about child selectors for even more targeted styling.
