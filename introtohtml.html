Introduction to HTML - Lesson Summary

This course is based on the Goodell Group HTML Tutorial.

  • Day One - June 12, 2020 (Chapters 1 to 5)

    Today we learned about the definition of HTML and the basic structure of an HTML page. We also learned about tags and were able to create our first page.

  • Day Two - June 19, 2020 (Chapters 6 to 7)

    We reviewed last week's lesson, then we learned about ordered and unordered lists, as well as adding links to other web pages/websites using the "a" tag.

  • Day Three - June 26, 2020 (Chapters 9 to 12)

    We learned about adding images to web pages, both from an internal image file as well as external links. We also tried out different styling tags and basic inline styling.
